Recursos para aprender desarrollo web

Learning web development is easier today thanks to many high-quality online resources. You can find free courses, detailed documentation, books, and video tutorials for all levels. Below you will find a table with some recommended resources and a list with a few personal favorites.

Web development resources

Resource name Type Link
MDN Web Docs Documentation / Reference https://developer.mozilla.org/
freeCodeCamp Interactive course https://www.freecodecamp.org/
The Odin Project Full web dev curriculum https://www.theodinproject.com/
W3Schools Tutorials https://www.w3schools.com/
Traversy Media (YouTube) Video tutorials https://www.youtube.com/c/TraversyMedia

My 3 favorite resources

  • MDN Web Docs – The best reference for HTML, CSS, and JavaScript.
  • freeCodeCamp – Excellent free interactive exercises and certifications.
  • The Odin Project – A structured, project-based path for full-stack web development.
